Al ejecutar el comando node server.js, se inicia el servidor de la aplicación y queda listo para recibir solicitudes desde el navegador o desde otros servicios.

En la terminal aparece un mensaje indicando que el servidor se encuentra activo y escuchando conexiones. Por ejemplo: “Servidor iniciado correctamente en el puerto 3000”.

Cuando se visita la ruta /estado, el servidor responde con información básica sobre el funcionamiento de la API, confirmando que el servicio está disponible.

Ejemplo de respuesta:

{
“estado”: “Activo”,
“mensaje”: “La API está operando correctamente”
}
